# iss-tracker-to-chat
An ISS Tracker that posts the location of ISS to Twitch chat along with a random word and translation of the country it is above.
Requirements:
 * requests library

Required environment variables:
 * GOOGLE_API_KEY
 * RADAR_API_KEY
 * TWITCH_NICK
 * TWITCH_OAUTH
 * TWITCH_CHAN
 * YANDEX_API_KEY
